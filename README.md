This is a PCB that helps you rebuild your failed Delta SMP-220DB Power Supply, found in your Macintosh Quadra 840AV or Power Macintosh 8100, with easily obtainable second hand parts.

Obtain an Antec EarthWatts PSU, one of the early ones that uses Seasonic internals, and mount the extracted PCB onto this new design using 16mm standoffs.
