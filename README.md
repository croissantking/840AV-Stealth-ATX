This is a PCB that helps you convert your failed Delta 220DB Power Supply, found in your Macintosh Quadra 840AV or Power Macintosh 8100, into a Stealth ATX unit.

You will need to obtain a used Antec EarthWatts PSU, one of the early ones that uses Seasonic internals, and mount the extracted PCB onto this new design using 16mm standoffs.
