This is a PCB that helps you rebuild your failed Delta SMP-220DB Power Supply, found in your Macintosh Quadra 840AV or Power Macintosh 8100, with easily obtainable second hand parts.

This design accepts an Antec EarthWatts PSU, one of the early ones that uses Seasonic internals, whose PCB should be extracted from its casing and mounted onto the four attachment holes using 16mm standoffs.
